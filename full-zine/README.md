# 팬게이머 팝업 ZINE 전체 원고 페이지

## 구성
- `index.html`: 페이지 본문
- `styles.css`: 반응형 디자인
- `script.js`: 게임 선택 및 모바일 목차
- 전체 원고는 `ZINE_문체_통일본.docx`를 기준으로 반영했습니다.

## 배포 방법 — GitHub Pages
1. 새 GitHub 저장소를 만듭니다.
2. 이 폴더 안의 파일 3개를 저장소 최상단에 업로드합니다.
3. 저장소 Settings → Pages로 이동합니다.
4. Deploy from a branch, `main` / `(root)`를 선택합니다.
5. 생성된 URL을 QR 코드로 변환합니다.

## 수정 포인트
- 메인 문구: `index.html`의 “더 많은 이야기가 보고 싶으신가요?”
- 하단 표기: `index.html`의 `<footer>`
- 색상: `styles.css` 상단의 `:root`
- 장면 코너는 스포일러 방지를 위해 접힌 상태로 표시됩니다.
